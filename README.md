# Next-Quiz

Quiz web application made with Next.js.

## Features

- Random choices
- Random questions
- Timer

## Technologies

- [NextJS][next]
- [Sass][sass]
- [Typescript][typescript]
- [React Countdown Circle Timer][react_timer]
